# 노트북에서 실행하기 (AI 사전 QA 자동화 시스템 · 식품)

## 사전 준비 (한 번만)
- **Python 3.10+** : https://www.python.org (설치 시 "Add to PATH" 체크)
- **Node.js 18+** : https://nodejs.org

## 1) 최초 설치
`setup.bat` 더블클릭 → 가상환경 생성 + 백엔드/프런트 의존성 설치
(easyocr·torch 포함이라 수 GB·수 분 소요)

## 2) 실행
`run.bat` 더블클릭 →
- 백엔드(FastAPI) http://127.0.0.1:8800
- 프런트(웹) http://localhost:5173 ← 자동으로 브라우저가 열림
- 로그인 : **admin / test1234**

창(검은 cmd 2개)을 닫으면 서버가 종료됩니다.

## 주의
- **품질검토 판정**은 원격 Gemma 서버(`222.110.207.7:8000`)에 접속되어야 동작합니다.
  사내망/VPN 등으로 그 서버에 닿는 환경에서 실행하세요. (안 닿으면 추출·판정이 실패)
- 식약처(식품안전나라)·법제처 OpenAPI는 인터넷 연결만 있으면 됩니다.
- HWP/HWPX 서류 파싱은 `kordoc`(Node CLI)이 필요합니다. setup.bat 이 자동 설치를 시도합니다.

## 수동 실행(배치 대신)
```
:: 백엔드
set QR_ADMIN_USER=admin & set QR_ADMIN_PASS=test1234 & set QR_JWT_SECRET=dev-secret
.venv\Scripts\python.exe -m uvicorn chandra_api.app:app --host 127.0.0.1 --port 8800

:: 프런트 (다른 창)
cd frontend && npm run dev
```
